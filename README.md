# Teslo API

1. Clonar Proyecto

2. ```yarn install```

3. Clonar archivo ```.env.template``` y renombrar a ```.env```

4. Setear variables de entorno

5. Levantar DB
```
docker-compose up -d
```
6. Ejecutar Seed
```
http://localhost:3000/api/seed
```
7. Levantar app
```
yarn start:dev
```
